Installation

This guide will walk you through the installation process and prerequisites to get your admin dashboard up and running smoothly.

  • Install PHP:
    • If you're using XAMPP, WAMP, or MAMP, PHP should already be included. Make sure your server environment (Apache, Nginx, etc.) is configured to work with PHP. If PHP is not installed, you can download it from the official PHP website: PHP Downloads.
  • Install Composer:
    • We recommend installing Composer globally. If it is not already installed, you can download and install Composer from the official website: Composer.org.
  • Install Symfony CLI:
    • Symfony CLI is a tool that helps you develop applications locally.
    • Download and run the installer from Symfony CLI official.
  • Download and Extract the Zip File:
    • Download and extract zip file, then navigate to the extracted directory.
  • Install Dependencies:
    • Run the following command to install the required PHP packages using Composer
    • composer install
  • Start the Symfony Server
    • Run the following command to start the Symfony local server
    • symfony server:start
    • If you wish to run the project locally on different Port.
    • symfony server:start --port=8001
  • Open Your Application in a Browser
    • Open a web browser and navigate to
    • http://127.0.0.1:{PORT}
